SUUMO 築年数表示 プライバシーポリシー

1. 収集する情報とその利用目的 本拡張機能「SUUMO 築年数表示」は、不動産情報サイト「SUUMO」のウェブページ上に表示されている「完成時期（築年月）」のテキストデータを、ユーザーのブラウザ上でのみ読み取ります。 読み取ったデータは、現在の「築年数」を自動計算し、画面上に表示するためにのみ使用されます。

2. データの送信・保存について 本拡張機能は、読み取ったデータやユーザーの個人情報を外部のサーバーへ送信したり、データベースに保存したりすることは一切ありません。すべての処理は、ユーザーのブラウザ（ローカル環境）内でのみ完結します。

3. 第三者への提供 本拡張機能は、ユーザーのデータを第三者に提供・開示することはありません。

4. アクセス解析・トラッキング 本拡張機能には、Google Analytics等のアクセス解析ツールや、ユーザーの行動を追跡するトラッキングコードは含まれていません。
